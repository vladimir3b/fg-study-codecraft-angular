Here we use ngNonBindable, thus {{ message }} = {{ message }}
Structural directive ngIf in example: 
<div 
  class="alert alert-info"
  *ngIf="state">
  Now you see me!!!
</div>
is equivalent to 
<ng-template [ngIf]="state">
  <div class="alert alert-info">
    Now you see me!!!
  </div> 
</ng-template>


Thus:
Now you see me!!!
is the same thing to
Now you see me!!!
Structural directive ngFor in example: 
<ul>
  <li *ngFor='let name of names'>
    {{ name }}
  </li>
</ul>
is equivalent to 
<ul class="alert">
  <ng-template
    ngFor
    let-name
    [ngForOf]="names">
    <li>
      {{ name }}
    </li>
  </ng-template>
</ul>

Thus:
  • {{ name }}
is the same thing to
  • {{ name }}
